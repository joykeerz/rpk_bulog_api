## API Setup

Untuk setup di local environment, mohon ikuti command berikut :

- git clone https://github.com/joykeerz/rpk_bulog_api.git
- composer update
- composer upgrade
- composer install
- php artisan migrate:fresh
- php artisan db:seed
- php artisan serve
